# Protocompat

This project aims to enumerate all possible scenarios for changes to a Protocol
Buffer schema. 

## Compatibility Matrix

- Messages
  - [x] Change a field type
  - [ ] Change a field name
  - [ ] Change a field tag
  - [x] Add a field
  - [ ] Remove a field
  - [x] Change a message name
- Enumerations
- Unions
- Services
